# Rocket OpenGL Simulation

Simulasi peluncuran roket sederhana pakai Python + OpenGL.

## Fitur
- Model roket 3D sederhana
- Thrust / gaya dorong
- Gravitasi
- Drag udara
- Massa berkurang karena bahan bakar terbakar
- Api dan asap partikel sederhana
- Kamera mengikuti roket

## Jalankan
```bash
pip install -r requirements.txt
python main.py
